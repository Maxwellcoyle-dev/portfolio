About

My coding journey started in 2019 when I was teaching at the University of Utah. I’ve always had a knack for computers, recognizing this, the admin asked me to help manage our department's learning management system. I was excited to help out but realized that if I wanted to make an impact, I would need to start learning HTML and CSS. The more I learned, the more I realized that I really enjoyed both learning and writing code, and that I was actually pretty good at it! A seed was planted.

My next step was to move from higher education to instructional design. Instructional design gave me an outlet to use a lot of the code that I was learning and I was able to push the bar using custom code solutions to learning and developments problems. In this role I also had an opportunity to learn JavaScript and to gain critical experience as a consultant.

Ultimately, it became clear to me that writing code was what I wanted to do. The next step was to take my JavaScript skills to the next level by learning React. Now, I am excited to start a new chapter of my career by working full-time as a front-end developer.
